# 民俗学データ更新作業ログ

## 作業日時
2025-11-09

## 作業概要
民俗学に関係のない観光地・建築物・歴史的人物などのスポットを削除し、神話・妖怪・伝承を中心とした民俗学データに入れ替えました。

---

## 作業の背景

当初のデータセットには、観光地・建築物・歴史上の人物など、民俗学とは直接関係のないスポットが多数含まれていました。本プロジェクトの目的である「民俗学版・大島てる」を実現するため、データを神話・妖怪・伝承・民間信仰を中心とした内容に絞り込みました。

```json
{

  "description": "詳細な説明（300〜500文字程度）",
  "address": "住所",
  "icon_type": "アイコン種別（ONI/KITSUNE/DRAGON/GENERIC等）",
  "era_hint": "時代（平安時代、江戸時代等）",
  "sources": [
    {
      "type": "出典タイプ（BOOK/URL/INTERVIEW）",
      "citation": "出典情報",
      "url": "URL（ある場合）"
    }
  ]
}
```

### 4. 新データのデータベース追加

#### 作成スクリプト
`scripts/add-new-folklore.ts`

#### 処理内容

1. JSONファイルから新規データを読み込み
2. 各スポットの住所をGoogle Maps APIでジオコーディング
3. 正確な位置情報（緯度・経度・Place ID）を取得
4. データベースに追加（status: PUBLISHED）

#### 実行コマンド

```bash
DATABASE_URL="postgresql://postgres:postgres@127.0.0.1:54322/postgres" \
  pnpm tsx scripts/add-new-folklore.ts
```

#### 結果

- 成功: 16件
- 失敗: 0件
- 実行時間: 約35秒（レート制限対策で2秒間隔）

## 今後の方針

### データ拡充の方向性

1. **各地の妖怪伝説**
   - 地域ごとの河童伝説
   - 各地の天狗伝説
   - 山姥伝説のバリエーション

2. **神社の由来譚**
   - 創建にまつわる神話
   - 神社の霊験譚
   - 狐や蛇など動物神の伝説

3. **怪談・幽霊話**
   - 各地の皿屋敷伝説
   - 落ち武者の霊
   - 怨霊信仰

4. **民間信仰**
   - 道祖神信仰
   - 庚申信仰
   - 地蔵信仰

5. **自然崇拝**
   - 巨木信仰
   - 滝信仰
   - 岩石信仰

### データ追加時の注意点

- 必ず出典を明記
- 民俗学的な意義があるものに限定
- 単なる観光地や建築物は除外
- 地域に根ざした伝承を優先
